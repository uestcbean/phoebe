## 📖 Phoebeについて

Phoebeは賢い子犬の名前から取られたインテリジェントなChromeブラウザ拡張機能です。インテリジェントなモード選択、タグ管理、多言語インターフェースサポートにより、貴重なWebコンテンツをNotionに素早く収集できます。

## ✨ 主な機能

### 🎯 インテリジェントモード選択 (v1.0.2 新機能)
- 📄 **ページモード**: 選択したページにコンテンツを直接追加、シンプルな収集に最適
- 🗄️ **データベースモード**: ページ選択や新規ページ作成をサポート、構造化された管理に最適

### 💫 ユーザーエクスペリエンスの最適化 (v1.0.2 新機能)
- 🎨 **ブランド化UI**: 美しいPhoebeロゴダイアログデザイン
- 🔄 **スマートローディング**: 作成・保存プロセスのリアルタイム状態表示
- 🛡️ **重複検出**: 重複ページ名の自動検出で混乱を回避
- ⚡ **重複操作防止**: 操作中のボタン自動無効化で安定性向上

### 🔧 コア機能
- 🔍 **スマート選択**: 任意のWebテキストを選択して右クリックで保存
- 📝 **メモ機能**: 収集したコンテンツに個人的なメモを追加
- 🏷️ **タグ管理**: スマートタグ提案と履歴
- 🌍 **多言語サポート**: 中国語、英語、日本語、韓国語、ドイツ語
- ⚡ **クイック同期**: Notionワークスペースに直接保存

## 🚀 インストール手順

1. `phoebe-v1.0.2.zip` ファイルをダウンロード
2. Chromeブラウザの拡張機能ページ (`chrome://extensions/`) を開く
3. 「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリックして、解凍したフォルダを選択
5. プラグイン設定でNotion APIキーと使用モードを設定

## ⚙️ 設定ガイド

### ステップ1: Notion統合の作成
1. [Notion統合ページ](https://www.notion.so/my-integrations) にアクセス
2. 「新しい統合」ボタンをクリック
3. 統合名を入力（例：Phoebe）
4. 関連するワークスペースを選択
5. 生成されたAPIキー（シークレットトークン）をコピー

### ステップ2: 使用モードの選択 (v1.0.2 新機能)

#### 📄 ページモード
- 適用場面: シンプルなコンテンツ収集、日記スタイルの追加
- 特徴: ターゲットページを選択、コンテンツはページの末尾に直接追加
- 利点: 簡単な設定、時系列でのコンテンツ整理

#### 🗄️ データベースモード
- 適用場面: 構造化されたコンテンツ管理、分類収集
- 特徴: データベースを選択、保存時にページを選択または新規作成
- 利点: 柔軟な管理、分類と検索をサポート

### ステップ3: スマート設定 (v1.0.2 新機能)
1. ブラウザのPhoebeプラグインアイコンをクリック
2. 「設定」ボタンをクリック
3. 「Notion APIキー」を入力
4. 使用モード（ページ/データベース）を選択
5. **スマート取得**: Phoebeが自動的にアクセス可能なページとデータベースリストを取得
6. リストからターゲットページまたはデータベースを選択
7. 「接続テスト」をクリックして設定を確認
8. 設定を保存

### 💡 権限設定のヒント
使用中にPhoebeが権限不足を示す場合：
1. 対応するNotionページまたはデータベースを開く
2. 右上の「共有」ボタンをクリック
3. 統合を検索して招待（例：Phoebe）
4. 「編集」権限を付与
5. プラグインに戻ってリソースリストを更新

## 📱 使用方法

### 基本的な使用方法
1. 任意のWebページで収集したいテキストを選択
2. 右クリックして「Notionに保存」を選択
3. 表示されるPhoebeダイアログで:
   - 選択したコンテンツを確認
   - メモを追加（オプション）
   - タグを追加（オプション）
   - **データベースモード**: ターゲットページを選択または新規ページを作成
4. 「保存」ボタンをクリック

### 🗄️ データベースモード特色機能
- **ページ選択**: データベース内の既存ページから選択
- **ワンクリック作成**: データベースに新しいページを素早く作成
- **重複検出**: 重複ページ名の自動検出とフレンドリーなプロンプト
- **リアルタイムローディング**: 作成プロセスで「Phoebeが一生懸命作成中...」を表示

### 📄 ページモード特色機能
- **直接追加**: プリセットページの末尾にコンテンツを自動追加
- **タイムスタンプ**: 各保存にタイムスタンプを追加
- **ページ情報**: 確認のためターゲットページ名を表示

### 🏷️ タグ管理のコツ
- **新しいタグの追加**: タグ入力ボックスに直接入力し、Enterで追加
- **履歴タグの選択**: 入力ボックスをクリックして履歴タグ提案を表示
- **タグ検索**: キーワードを入力してタグ提案をフィルタ
- **タグ削除**: 追加されたタグの隣の×ボタンをクリック

## 🚫 トラブルシューティング

### 接続に失敗した場合
1. APIキーが正しくコピーされているか確認
2. ページ/データベースIDの形式が正しいか確認（32文字の文字列）
3. 統合がターゲットリソースに招待されているか確認
4. 「編集」権限が付与されているか確認

### モード選択の推奨事項
- **新規ユーザー**: まずページモードを試用、シンプルで使いやすい
- **上級ユーザー**: より強力な機能のためにデータベースモードを使用
- **柔軟な切り替え**: 設定でいつでもモードを変更可能

### ページ作成に失敗した場合
1. 同じ名前のページが既に存在するか確認
2. データベース権限が正しく設定されているか確認
3. ネットワーク接続が安定しているか確認
