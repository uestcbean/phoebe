## 📖 Phoebe 소개

Phoebe는 똑똑한 강아지의 이름을 따서 만든 지능형 Chrome 브라우저 확장 프로그램입니다. 지능형 모드 선택, 태그 관리, 다국어 인터페이스 지원을 통해 소중한 웹 콘텐츠를 Notion에 빠르게 수집할 수 있습니다.

## ✨ 주요 기능

### 🎯 지능형 모드 선택 (v1.0.2 신기능)
- 📄 **페이지 모드**: 선택한 페이지에 콘텐츠를 직접 추가, 간단한 수집에 최적
- 🗄️ **데이터베이스 모드**: 페이지 선택 또는 새 페이지 생성 지원, 구조화된 관리에 최적

### 💫 사용자 경험 최적화 (v1.0.2 신기능)
- 🎨 **브랜드 UI**: 아름다운 Phoebe 로고 대화상자 디자인
- 🔄 **스마트 로딩**: 생성 및 저장 과정의 실시간 상태 표시
- 🛡️ **중복 검출**: 중복 페이지 이름 자동 감지로 혼란 방지
- ⚡ **중복 작업 방지**: 작업 중 버튼 자동 비활성화로 안정성 향상

### 🔧 핵심 기능
- 🔍 **스마트 선택**: 웹 텍스트를 선택하고 우클릭으로 저장
- 📝 **메모 기능**: 수집한 콘텐츠에 개인 메모 추가
- 🏷️ **태그 관리**: 스마트 태그 제안 및 히스토리
- 🌍 **다국어 지원**: 중국어, 영어, 일본어, 한국어, 독일어
- ⚡ **빠른 동기화**: Notion 워크스페이스에 직접 저장

## 🚀 설치 단계

1. `phoebe-v1.0.2.zip` 파일 다운로드
2. Chrome 브라우저의 확장 프로그램 페이지 (`chrome://extensions/`) 열기
3. "개발자 모드" 활성화
4. "압축해제된 확장 프로그램 로드"를 클릭하여 압축 해제된 폴더 선택
5. 플러그인 설정에서 Notion API 키와 사용 모드 설정

## ⚙️ 설정 가이드

### 1단계: Notion 통합 생성
1. [Notion 통합 페이지](https://www.notion.so/my-integrations) 방문
2. "새 통합" 버튼 클릭
3. 통합 이름 입력 (예: Phoebe)
4. 연결할 워크스페이스 선택
5. 생성된 API 키(비밀 토큰) 복사

### 2단계: 사용 모드 선택 (v1.0.2 신기능)

#### 📄 페이지 모드
- 적용 대상: 간단한 콘텐츠 수집, 일기 스타일 추가
- 특징: 대상 페이지 선택, 콘텐츠가 페이지 끝에 직접 추가됨
- 장점: 간단한 설정, 시간순으로 콘텐츠 정리

#### 🗄️ 데이터베이스 모드
- 적용 대상: 구조화된 콘텐츠 관리, 분류 수집
- 특징: 데이터베이스 선택, 저장 시 페이지 선택 또는 새로 생성
- 장점: 유연한 관리, 분류 및 검색 지원

### 3단계: 스마트 설정 (v1.0.2 신기능)
1. 브라우저의 Phoebe 플러그인 아이콘 클릭
2. "설정" 버튼 클릭
3. "Notion API 키" 입력
4. 사용 모드(페이지/데이터베이스) 선택
5. **스마트 가져오기**: Phoebe가 자동으로 접근 가능한 페이지와 데이터베이스 목록을 가져옴
6. 목록에서 대상 페이지 또는 데이터베이스 선택
7. "연결 테스트"를 클릭하여 설정 확인
8. 설정 저장

### 💡 권한 설정 팁
사용 중 Phoebe가 권한 부족을 알릴 경우:
1. 해당 Notion 페이지 또는 데이터베이스 열기
2. 우상단의 "공유" 버튼 클릭
3. 통합을 검색하여 초대 (예: Phoebe)
4. "편집" 권한 부여
5. 플러그인으로 돌아가서 리소스 목록 새로고침

## 📱 사용 방법

### 기본 사용법
1. 웹페이지에서 수집하고 싶은 텍스트 선택
2. 우클릭하여 "Notion에 저장" 선택
3. 나타나는 Phoebe 대화상자에서:
   - 선택한 콘텐츠 확인
   - 메모 추가 (선택사항)
   - 태그 추가 (선택사항)
   - **데이터베이스 모드**: 대상 페이지 선택 또는 새 페이지 생성
4. "저장" 버튼 클릭

### 🗄️ 데이터베이스 모드 특색 기능
- **페이지 선택**: 데이터베이스 내 기존 페이지에서 선택
- **원클릭 생성**: 데이터베이스에 새 페이지를 빠르게 생성
- **중복 검출**: 중복 페이지 이름 자동 감지 및 친화적 알림
- **실시간 로딩**: 생성 과정에서 "Phoebe가 열심히 생성 중..." 표시

### 📄 페이지 모드 특색 기능
- **직접 추가**: 미리 설정된 페이지 끝에 콘텐츠 자동 추가
- **타임스탬프**: 각 저장에 타임스탬프 추가
- **페이지 정보**: 확인을 위한 대상 페이지 이름 표시

### 🏷️ 태그 관리 팁
- **새 태그 추가**: 태그 입력 상자에 직접 입력하고 Enter로 추가
- **히스토리 태그 선택**: 입력 상자를 클릭하여 히스토리 태그 제안 보기
- **태그 검색**: 키워드를 입력하여 태그 제안 필터링
- **태그 삭제**: 추가된 태그 옆의 × 버튼 클릭

## 🚫 문제 해결

### 연결 실패 시
1. API 키가 올바르게 복사되었는지 확인
2. 페이지/데이터베이스 ID 형식이 올바른지 확인 (32자 문자열)
3. 통합이 대상 리소스에 초대되었는지 확인
4. "편집" 권한이 부여되었는지 확인

### 모드 선택 권장사항
- **신규 사용자**: 먼저 페이지 모드 시도, 간단하고 사용하기 쉬움
- **고급 사용자**: 더 강력한 기능을 위해 데이터베이스 모드 사용
- **유연한 전환**: 설정에서 언제든지 모드 변경 가능

### 페이지 생성 실패 시
1. 같은 이름의 페이지가 이미 존재하는지 확인
2. 데이터베이스 권한이 올바르게 설정되었는지 확인
3. 네트워크 연결이 안정적인지 확인
